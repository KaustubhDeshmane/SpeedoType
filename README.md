# 📖 Overview

SpeedoType is a clean and interactive typing speed test built with React and Vite. It provides instant feedback while typing, allowing users to monitor their performance through real-time statistics and visual character highlighting.

The project focuses on delivering a distraction-free typing experience while maintaining a modern UI with responsive layouts and theme customization.

---

# ✨ Features

- ⚡ Real-time Words Per Minute (WPM)
- 🎯 Live Accuracy Calculation
- 🏆 Best Score Tracking using Local Storage
- 🌗 Light & Dark Theme
- 📚 Easy, Medium & Hard Difficulty Levels
- ✍️ Live Character-by-Character Highlighting
- 🔄 Instant Restart
- 📱 Fully Responsive Design
- 🎨 Modern Glassmorphism Interface
- 🔊 Typing Sound Toggle
- ⌨️ Monkeytype-inspired Blinking Caret
- 🟢🟡🔴 Live Character Status Highlighting
- 💾 Persistent Best Score Storage

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/KaustubhDeshmane/SpeedoType.git
```

Navigate to the project

```bash
cd SpeedoType
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```

Preview production build

```bash
npm run preview
```

# 🎯 How It Works

1. Choose a difficulty level.
2. Start typing the displayed paragraph.
3. Monitor your live WPM and accuracy.
4. Complete the paragraph.
5. Beat your previous best score.
